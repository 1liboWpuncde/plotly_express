plotly_express

plotly_express is a terse, consistent, high-level wrapper around plotly for rapid data exploration and figure generation.

Sub-modules

plotly_express.colors

Built-in qualitative color sequences and sequential, diverging and cyclical color scales.

plotly_express.data

Built-in datasets for demonstration, educational and test purposes.
